How does APR work?

Automatic Program Repair is a field of study centered around automatically fixing bugs without requiring developer input.

Genetic Improvement Algorithms

When a test fails, we use genetic algorithms to search local and global code bases to detect fixes. Once these are found we provide a developer the option of three potential fixes to implement.

The FIXIE Project
Exploiting defect predicition for automatic software repair (FIXIE), is a software engineering research project funded by the EPSRC.
